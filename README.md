# 🎬 StreamFlix — DAT 1. Semester SP3

Et tekstbaseret streamingtjeneste-system bygget i Java,
udviklet som en del af SP3 projektet på CPH Business.

## 📖 Om projektet
StreamFlix simulerer en streamingplatform som Netflix eller HBO.
Brugere kan søge, gemme og "afspille" film og serier — alt fra terminalen.

## ✨ Funktioner
- 🔐 Brugeroprettelse & login (gemt i fil)
- 🔍 Søg efter film via titel eller kategori
- 📋 Personlig seeliste & afspilningshistorik
- ▶️ "Afspil" film og serier via terminalen
- 📁 Data indlæses fra .txt filer

## 🏗️ Bygget med
- Java (OOP — klasser, interfaces, ArrayLists)
- File I/O til vedvarende data
